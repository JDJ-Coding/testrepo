설정 파일이 왜 두개인가요?

.claude.json은 MCP 같은 도구 설정이며,
.claude/settings.json은 명령어 실행 권한 같은 보안 설정입니다.

⚠️

웹에서 로컬 MCP가 안 돼요!

Claude.ai 웹은 샌드박스 환경이라 로컬 터미널(stdio) 방식의 MCP는 지원하지 않습니다. 오직 외부에서 접근 가능한 HTTP 서버 방식만 연결 가능합니다.
