💡 FAQ

  • Q. Claude Desktop과 Code의 MCP는 공유되나요? 절대 아닙니다. 각각 별도의 설정 파일에 등록해야 합니다.
  • Q. CLAUDE.md는 어디에 두나요? - 전역: ~/.claude/CLAUDE.md
    - 프로젝트: [프로젝트]/CLAUDE.md
  • Q. Cowork 에러 발생 시? Claude-재시작.ps1을 실행하여 서비스를 재설정하세요.

🎯 핵심 요약

• Desktop MCP: claude_desktop_config.json

• Claude Code MCP: ~/.claude.json (+ .mcp.json)

• CLI ↔ VS Code 확장: 같은 설정을 공유하는 한 몸입니다.

웹 MCP는 원격 HTTP만 지원하며, 로컬 환경(stdio)은 데스크탑 앱이나 Claude Code를 통해서만 가능합니다.
